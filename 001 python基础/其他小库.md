##### 1、bisect

bisect是python内置模块,用于有序序列的插入和查找

- 查找：bisect(array,item)

- 插入：insort(array,item)

  查找

  ```python
  # *-* coding:utf-8 *-*
  import bisect
  a= [1,3,5,7,9,11]
  position = bisect.bisect(a,2) #查找元素2在a列表中个位置
  print(position) # 1  
  
  bisect.insort(a,4)
  print(a)  #[1, 3, 4, 5, 7, 9, 11]
  ```

##### 2、apscheduler

inux 自带的 crontab ，windows 自带的任务计划，都可以实现定时任务。如果需要更加精细化的控制，或者说任务程序需要跨平台运行，Python 的 apscheduler 提供了非常丰富而且方便易用的定时任务接口。

apscheduler 不是一个守护进程或服务，它自身不带有任何命令行工具。它主要是要在现有的应用程序中运行，也就是说，apscheduler 为我们提供了构建专用调度器或调度服务的基础模块。

[参考](https://blog.csdn.net/somezz/article/details/83104368)

```python
# *-* coding: utf-8 *-*
import datetime

from apscheduler.schedulers.blocking import  BlockingScheduler

def tick():
    print ('Tick! the time is: %s'%datetime.datetime.now())


if __name__ == '__main__':
    scheduler = BlockingScheduler() #最简单的调度器
    scheduler.add_job(tick,'interval',seconds=3)  #添加作业，触发器为interval
    try:
        scheduler.start()
    except :
        pass
```

##### 3、imp

python根据路径导入模块的方法

```

#方法一、
```

python解释器查找模块文件的过程
通常情况下，当使用import语句导入模块后，python会按照以下顺序查找指定的模块文件

- 在当前目录，即当前执行的程序文件所在目录下查找
- 到PYTHONPATH(环境变量)下的每个目录中查找
- 到python默认的安装目录下查找(lib/site_packages/)

##### 4、configparser ---配置文件解析器

此模块提供了它实现一种基本配置语言ConfigParser类，这种语言提供的结构与 ini文件的类似。

ini文件：这种文件由多个节组成，每个节包含多个带有值的键。configparser类可以读取和写入这种文件。

- 通过程序创建ini配置文件

  ```python
  import configparser
  config = configparser.ConfigParser()
  config['section1'] ={'a':1,'b':2}
  config['section2'] = {'c':3,'d':4}
  
  with open('b.ini','w') as f :
      config.write(f)
   
  '''
  b.ini文件
  
  [section1]
  a = 1
  b = 2
  
  [section2]
  c = 3
  d = 4
  '''
  ```

- 读取ini文件

  ```python
  import configparser
  
  config = configparser.ConfigParser()
  config.read('b.ini')
  print (config.sections())  # [u'section1', u'section2']
  
  section1 = config['section1'] 
  print(section1['a'])   
  #或者 
  print(config.get('section1','a'))
  ```
