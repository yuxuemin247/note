# 伪类选择器

伪类的语法 ：

选择器：pseudo-class { `property:value` }

伪类可以配合`css`类使用



### 1、a标签四大伪类

- :link：未访问状态

- :hover：悬浮状态

- :active：活跃状态(在被点击的时候)

- :visited：已访问状态

  ```
  <html>
  <head>
  
  <style type="text/css">
  a:link {color: #FF0000}
  a:visited {color: #00FF00}
  a:hover {color: #FF00FF}
  a:active {color: #0000FF}
  </style>
  </head>
  <body>
  <p><b><a href="/index.html" target="_blank">这是一个链接。</a></b></p>
  <p><b>注释：</b>在 CSS 定义中，a:hover 必须位于 a:link 和 a:visited 之后，这样才能生效！</p>
  <p><b>注释：</b>在 CSS 定义中，a:active 必须位于 a:hover 之后，这样才能生效！</p>
  </body>
  </html>
  ```

### 2、内容伪类

- ：focus 获取焦点
- ::selection 选中的元素

- :before：内容之前
- :after：内容之后

```css
:before, :after {
    content: "ctn";
}
```

### 3、索引伪类

- :nth-child(n)：位置优先，再匹配类型，选择第n个子元素
- :nth-of-type(n)：类型优先，再匹配位置

```
p:first-child的意思不是选中p元素的第一个子元素，而是选择所有作为某个元素的第一个子元素的p
```

###### v_hint：值可以为位置数，也可以为2n、3n...，代表2的倍数，3的倍数，且位置数从1开始

### 4、取反伪类

- :not(selector)：对selector进行取反	

  div:not(.class3)  选择所有不包含class3的元素

- pre:not(:empty) 选择含有子元素的pre元素

  :empty选择器匹配没有子元素(包括文本节点)的每个元素
