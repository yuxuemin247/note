# W3C知识点运用

### 1、页面基础复习

```html
<link rel="shortcut icon" type="image/x-icon" href="bd.ico" >
<style type="text/css">
    .d1 {
        width: 200px;
        height: 200px;
        background-color: red;
    }
    /*文本*/
    .d1 {
        font: 900 50px/200px 'STSong';
        text-align: center;
        color: orange;
        letter-spacing: 10px
    }
    /*背景图片*/
    .d1 {
        background: #ccc url('bd.ico') no-repeat center center;
    }
    /*边框*/
    .d1 {
        border: 5px solid cyan;
    }
    /*边界圆角*/
    .d1 {
        border-radius: 10px 20px 30px / 40px 50px;
    }
</style>

<div class="d1">文本</div>
```

### 2、display之盒模型基本样式

```html
<style type="text/css">
    div {
        display: block;
        width: auto;
        height: 0;
    }
    span {
        display: inline;
        /*不支持*/
    }
    img {
        display: inline-block;
        width: auto;
        height: auto;
    }
</style>
<div></div>

```
