# JQ初级

## 一、认识jQuery

#### 1、什么是jQuery

- jQuery是对原生JavaScript二次封装的工具函数集合

- jQuery是一个简洁高效的且功能丰富的JavaScript工具库，js类库

  ```
  js类库就是一些函数的集合，就是把特定效果的代码写好，在用的时候用很少的代码去调用
  常见JS类库还有ExtJS,prototype.js,zepto.js
  ```

#### 2、jQuery的优势

- 完全开源的源代码
- 强大简洁的选择器
- 事件、样式、动画的良好支持
- 链式表达式
- 简化的Ajax操作
- 跨浏览器兼容
- 丰富的插件及对外的扩展接口

## 二、jQuery的安装

- 生产(production)版本：jQuery-x.x.x.min.js

## 四、功能概括

#### 1、选择器

```js
var $ele = $('.ele');  //DOM元素的获取，通过选择器来获取
```

#### 2、内容操作

```
$ele.text();  //获取所有匹配元素内部文本
$ele.html() //获取第一个匹配元素内部的html结构
$ele.html('<h2>你好<h2>') // 将字符串按照html格式解析设置到标签上
$ele.text('<h2>你好<h2>')  //将字符串按照普通文字设置到标签上
$ele.val(); //获取表单的值
$ele.val('打游戏') // 给表单值设置值
```

#### 3、属性操作

```
$(选择器).attr('属性名')   //返回被选元素的属性值
$(选择器).attr('属性名',val|fn)  // 新增修改获取属性值
$(选择器).removeattr('属性名')  //移除属性值
$(选择器).prop('属性名')  // 获取在匹配的元素集中的第一个元素的属性值
$(选择器).removeProp('属性名')   //移除属性值
//处理固有属性，如checked selected 要用 prop方法操作才能获得正确的结果
//prop 获取在匹配的元素集中的第一个元素的属性值
```

#### 4、样式操作

```js
$ele.css({width: '200px', heigth: '200px'});  //参数1代表样式，参数2代表值
$ele.css('background-color', 'red').css('border-radius', '50%');
```

#### 5、类名操作

```js
$ele.addClass('active');  //给选中元素添加类名
$ele.removeClass('active');  // 给选中的元素删除指定的类名
$ele.toggleClass('active');   // 给选中的元素切换时否有某个类名
$ele.hasclass('类名')；   //检测当前的元素是否包含某个特定的类
```

#### 6、元素位置

```
offset()		获取匹配元素在当前视口的相对偏移
position()		获取匹配元素相对父元素的偏移
scrollLeft()	获取匹配元素相对滚动条顶部的偏移
scrollTop()		获取匹配元素相对滚动条左侧的偏移	
widht()			取得第一个匹配元素当前计算的宽度值（px）
height()		取得匹配元素当前计算的高度值（px)
innerWidth()	匹配元素内部区域宽度（包括补白、不包括边框）
innerHeight()	匹配元素内部区域高度（包括补白、不包括边框）
outerWidth()	匹配元素外部宽度（默认包括补白和边框）
outerHeight()	匹配元素外部高度（默认包括补白和边框）
```

## 六、Ajax

- server.py

```python
from flask import Flask, request
from flask_cors import CORS
app = Flask(__name__)
CORS(app, supports_credentials=True)

@app.route('/testAction')
def test_action():
    usr = request.args['usr']
    ps = request.args['ps']
    if usr != 'zero' or ps != '123456':
        return 'login failed'
    return 'login success'

if __name__ == '__main__':
    app.run()
    
// 安装Flask及Flask-Cors包
```

- client.html
