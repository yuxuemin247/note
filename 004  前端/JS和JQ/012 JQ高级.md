# JQ高级

## 一、事件

1、on绑定

```js
$('.box').on('click', {num: 10}, function(ev) {
    console.log(ev.data.num)
})
// 数据由ev.data存储
```

2、非on事件

```js
$('.box').click({num: 10}, function(ev) {
    console.log(ev.data.num)
})
// 数据由ev.data存储
$('.box').click(function(){
    console.log('点了盒子')
})
```

3、解绑

```js
$('.box').off()
```

## 二、节点操作

1、内部插入

```js
append(content|fn)  //向选中的节点内部追加内容
appendTo(content)   //向选中的节点内部前置内容
prepend(content|fn) //把选中的元素追加到content内部
prependTo(content)  //把选中的元素前置到content内部
```

2、外部插入

```js
after(content|fn)   //向选中的标签外部后方插入元素content
before(content|fn)	//向选中的标签外部前方插入元素content
insertAfter(content) //将选中元素插入到content元素的外部后方
insertBefore(content)  //将选中元素插入到content元素的外部前方
```

3、包裹

```js
wrap(html|ele|fn)  //把所有选择的元素用其他字符串代码包裹起来，html参数为字符串代码用于生成元素，并且包裹所选元素
unwrap()
wrapAll(html|ele)
wrapInner(html|ele|fn) //用html生成一个双标签把选中的元素内部包裹起来，生成标签作为选中的标签的子标签
```

4、替换

```js
replaceWith(content|fn) //将所有的选择的元素替换成指定的元素或者DOM对象，其中参数为替换内容
replaceAll(selector)   //该方法是将selector选中的内容替换成函数前面的内容
```

5、删除

```js
empty()  //清空
remove([expr])  //其中参数expr是可选项，代表删除的条件
detach([expr])  //其中参数expr是可选项，代表删除的条件
```

6、复制

```js
clone([Even[,deepEven]])   //clone 的时候默认只克隆标签及其属性和内容，不克隆行为；如果想要克隆行为，在clone函数中加一个参数true
```

7、元素的遍历

```
//即使我们在用jquery的选择器选中的元素是一个可迭代的对象，也不能用下标访问其内部元素，用each来迭代
$("p").each(function(index){
    console.log(index)
    // 如果要单独操作某个元素，只需用if语句控制下标
    if (index==2) {
    $(this).css("color","red")
    // this代表当前方法中选中的元素
    }
})
```
