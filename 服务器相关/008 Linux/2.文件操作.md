## 目录操作

```python
当前路径
pwd

切换目录
cd .. 上一级目录   
cd /etc  切换指定目录

显示目录
ls
ll ==> ls -l
tree

创建目录
mkdir -p /t1/t2/t3

创建文件
touch 1.txt

查看文件
cat 1.txt

输出数据
echo "abc"

复制
cp /etc/1.txt  /tmp

剪切
mv /etc/1.txt  /tmp

删除
rm -rf /etc/1.txt
```

## 编辑模式

```python
vim 1.txt
i	编辑
:wq 保存  :q 退出   :q! 强退

```
