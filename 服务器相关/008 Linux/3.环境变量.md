## 字符编码

```python
查看编码
echo $LANG

临时修改
export LANG=zh_CN.UTF-8		

永久修改
1.修改配置文件
vim /etc/locale.conf

2.命令行修改
localectl set-locale LANG=en_US.UTF-8

source /etc/locale.conf 
```
