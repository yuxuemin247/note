## Flask项目

## 一、**项目分析（需求）**

博客 blog

1. 登录注册
2. 个人中心
   + 个人信息
     + 查看
     + 修改
   + 修改密码
   + 修改邮箱
   + 修改头像
   + 我收藏的博客
   + 我发表的
     + 查看
     + 修改
   + 评论我发表的博客
   + 回复了我的评论
3. 发表博客
   + 插件
4. 博客首页展示
   + 默认按照点击量(时间的排序)
   + 搜索
   + 轮播图
   + 所有博客的展示
   + 分页
5. 详情页
   + 博客的内容
   + 谁发表的
   + 发表时间
   + 浏览量
   + 收藏
   + 评论
   + 回复
