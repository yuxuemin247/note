###### 1、 编辑配置文件

```
编辑配置文件/etc/vsftpd/vsftpd.conf出错，出现以下提示： 
E325: ATTENTION 
Found a swap file by the name “/etc/vsftpd/.vsftpd.conf.swp
原因：可能是由于上一次在编辑vsftpd.conf时，文件没有退出就直接（强行）重启或关机造成的，产生.vsftpd.conf.swp文件。 
解决方法：在vsftpd.conf文件所在的目录，执行ls -a，会发现该文件，删除该文件—- rm /etc/vsftpd/.vsftpd.conf.swp
```

